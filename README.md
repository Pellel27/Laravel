# Lavavel 9

Projet de site d'un restaurant construit avec Laravel 9.
## Prérequis
-PHP 8.1
-MariaDB ou MYSQL
-Composer 2.3.5
nodejs 18.4.0 et npm 8.12.0

## Installation

```bash
composer install
npm install
```


## Utilisation

Dans un autre terminal, lancez :
```bash
Php artisan serve
```

Dans un autre terminal, lancez :
```bash
npm run dev
```
Pour voire la page d'accueil, ouvrire : [ Accueil](http://127.0.0.1:8000/)
